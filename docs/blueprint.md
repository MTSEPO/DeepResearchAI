# **App Name**: DeepResearchAI

## Core Features:

- AI-Powered Research Assistant: Provides real-time AI-driven competitive intelligence and insights for sales teams.
- User Authentication: Secure user authentication via email/password and OAuth providers (Google, Facebook).
- Pricing and Subscription Management: Offers flexible pricing tiers (PWYW, Hobbyist, Professional) and a Lifetime Deal (LTD) option with one-time payment.
- LTD Purchase Management: Enables users to purchase a Lifetime Deal, securely stores the purchase status in Firestore, and automatically grants lifetime access to premium AI features.
- Welcome Statement with Picture Holder: Create a Welcome Statement explaining the app, its benefits, and the PWYW model, including a placeholder for the user's profile picture and link to the user's page.
- Competitive Analysis Report: Leverage a generative AI tool to compile the findings on potential market gaps that exist between you and your competitors.
- Home/Landing page for SaaS Review: Landing page with information and links so that Lemon Squeezy can approve my app.

## Style Guidelines:

- Primary color: Sky Blue (#7DD3FC) to evoke a sense of trust, innovation, and intelligence. This is inspired by the original TsepoAI logo.
- Background color: Light gray (#F0F4F8), nearly white, for a clean and professional feel.
- Accent color: Light Blue (#90D7FF). This complements the Sky Blue, drawing the eye to important interactive elements such as buttons.
- Body and headline font: 'Inter', a sans-serif font, known for its clean readability and modern aesthetic. A versatile font choice for all content.
- Code font: 'Source Code Pro' for displaying code snippets, when/if they are displayed.
- Use modern and minimalistic icons. Icons related to data analysis and AI.
- Clean, modern layout with a focus on usability. Implement a grid-based system to ensure a consistent and professional look.
